# MF Character Detail

Microfrontend que muestra el detalle de un personaje de Rick & Morty.

## Responsabilidades

- Recibe el id del personaje desde el shell
- Muestra información ampliada del personaje
- Lista todos los episodios donde aparece

## Puerto

- http://localhost:3002

## Tecnologías

- React 18
- TypeScript
- Webpack Module Federation
- Tailwind CSS

## Levantar en desarrollo

npm install
npm start
