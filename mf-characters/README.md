# MF Characters

Microfrontend que muestra el listado de personajes de Rick & Morty.

## Responsabilidades

- Consume la API de Rick & Morty
- Muestra personajes en grid con paginación
- Permite filtrar por nombre, estado o especie
- Navega al detalle de cada personaje

## Puerto

- http://localhost:3001

## Tecnologías

- React 18
- TypeScript
- Webpack Module Federation
- Tailwind CSS

## Levantar en desarrollo

npm install
npm start
